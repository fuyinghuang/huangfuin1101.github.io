It is an e-commerce app where users can search by keywords, browse all the items and create purchase orders when the stock is sufficient. When user logs in as a administrator , items can be added, updated, deleted and the stock amount can be adjusted. Also, all the usersâ€™ purchased items can be traced.

Aldojo

Aldojo is a travelling info app created by Alex(Al), Doris(Do) and Josh(Jo) aiming for the traveler to use as a tour guide. Users have access to a variety of different features of the app when register and log in, such as adding more countries, creating, editing and deleting food on a particular country. The user can also review the country, food and even vote for the food. On the profile page, the user can see the record of the food and the country they have reviewed. It was my third project of Web Development Immersive course at General Assembly's. It was a group project built on 7 days with two colleagues and it also was my first go of MEAN stack app technologies.

My Hand-Making Diary

My Hand-Making Diary is my second individual project from General Assembly's Web Development Immersive. This project was a RESTful personal blog built with the first dive into backend technologies by Node.js, Express, EJS and Mongoose. My Hand-Making Diary is a hand-made brand I've been managed for years. This app is a blog with Pinterest style that user can sign up to edit or add more hand- made product and also can give the review for each item. The aim for this app is for hand-made lovers to share, to learn and to communicate.
